## Install / Run / Build

```shell
# install dependencies
yarn

# run dev server (open index.html in browser)
yarn run dev

# generate dist folder files
yarn build
```

## Publish to NPM
```shell
# login to npm
npm login

# publish
npm publish --access=public
```
